# Sentiment-Analysis-on-Social-Media
Este proyecto analiza el sentimiento en redes sociales utilizando técnicas de Procesamiento de Lenguaje Natural (NLP).  El objetivo es comprender cómo las personas expresan opiniones y emociones en datos de texto.
# 📊 Sentiment Analysis on Social Media | Análisis de Sentimiento en Redes Sociales

---

## 🇺🇸 English Version

## 📌 Project Overview

This project analyzes sentiment in social media data using Natural Language Processing (NLP).

The goal is to understand how people express opinions and emotions about social topics through text data.

---

## 🧠 Objectives

* Analyze sentiment in tweets
* Classify text into positive, negative, and neutral
* Visualize public opinion trends

---

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* TextBlob (NLP)
* Google Colab

---

## 📂 Dataset

Source: Twitter dataset from Kaggle
Data includes real user-generated text (anonymized).

---

## 📈 Key Insights

* Social media conversations show a mix of positive and negative sentiment
* Certain topics generate strong polarization
* Text data provides valuable insights into public perception
